Back to basis

To have a working API, just populate the app directory.

  • app/models will contains model classes.
  • app/controllers will contains controller classes.
  • app/views will contains custom views.

Usages

Beware the name of the files MUST follow some rules.

  • Controllers must be named NameController.rb
  • Views must be named TypeView.rb
  • Then you can access your controller using the URI: http://host/names/id.type.

For example, creating a UserController.rb in app/controllers will provide the URI: http://host/users/johndoe.json

More precisely, calling this URI will execute the UserController::show with @request[:user]="johndoe". Then the returned object is given to the JSONView provided by Rubyzome.
